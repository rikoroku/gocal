# Goedule
## Status
WIP

```
Goedule is a schedule management tool of Google Calendar

Options:

  -h, --help      display help information
  -v, --version   display version

Commands:

  help     display help of command information
  list     display today's schedules(aliases l)
  insert   creates a new schedule(aliases i)
  delete   deletes a schedule of specified id(aliases d)
```
